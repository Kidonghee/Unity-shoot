# Unity Shoot Game

간단한 2D 슈팅 게임입니다.  
플레이어가 적을 피하고 공격하면서 코인을 획득하고 무기를 업그레이드하는 게임입니다.

---

## 프로젝트 소개

- Unity로 제작한 2D 슈팅 게임
- 적을 처치하며 코인을 획득
- 일정 코인마다 무기 업그레이드
- 보스 등장 시스템 포함

---

## 개발 환경

- Unity
- C#
- Visual Studio

---

## 주요 기능

### 플레이어
- 마우스 이동으로 좌우 이동
- 자동 공격 시스템

### 무기 시스템
- 일정 시간마다 자동 발사
- 코인 획득 시 업그레이드

### 적 시스템
- 랜덤 위치에서 적 생성
- 점점 강해지는 난이도

### 보스 시스템
- 일정 조건 달성 시 등장
- 일반 적보다 강한 체력

### 코인 시스템
- 적 처치 시 코인 드롭
- 코인 획득 시 점수 증가

### 게임 오버
- 적 또는 보스와 충돌 시 게임 종료

---

## 실행 방법

1. Unity Hub 실행
2. 프로젝트 열기
3. SampleScene 실행
4. Play 버튼 클릭
